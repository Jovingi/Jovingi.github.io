Screen Space Global Illumination

Project information

  • Introduction

    • A real-time implementation of Screen Space Global Illumination (SSGI) using C++ and DirectX 11, developed as part of my final dissertation at Sheffield Hallam University.

    The effect is designed to enhance realism by simulating indirect lighting based on screen-space data in a deferred rendering pipeline.

  • What I did

    • In this project I implemented a complete SSGI pipeline on top of my deferred renderer, including screen-space position reconstruction, stochastic sampling, ray marching, and both temporal and spatial denoising.

    I also created an ImGui debug menu to visualize all key rendering stages such as the scene with direct lighting, stochastic normals, and the blurred indirect lighting result. The implementation was inspired by GameHacker's SSGI blog post.
